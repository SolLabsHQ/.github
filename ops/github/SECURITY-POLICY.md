# GitHub Protection Policy (SolLabsHQ)

## Branch protection target
- refs/heads/main

## Enforcement
- Pull requests required (no direct pushes to main)
- Required approvals: 1
- Dismiss stale approvals on new pushes
- Require conversation resolution before merge
- Block force pushes
- Block deletions

## Status checks
- Optional (enable later when CI exists)

This policy is enforced via GitHub Rulesets when available.
